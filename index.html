INDMoney TradeReplay™

TradeReplay is a learning sandbox for US stocks. It replays stylised price paths, compares simple strategies, and visualises stress scenarios so users can practice before committing real capital.

Concept built to demonstrate PM thinking Static front‑end · safe mock data · GitHub deployable

This is a static prototype to showcase how I would think about an educational, revenue‑linked trade replay experience for INDMoney. No live prices, no real trades, and no investment advice.
