INDMoney TradeReplay™ · Concept demo

This folder contains a static, front-end only prototype showcasing how an AI-inspired
trade replay & learning tool could look for INDMoney's US stocks product.

Files:
- index.html  : main layout + tab structure
- style.css   : glassmorphic, responsive UI
- script.js   : tab logic, mock data generation, Chart.js integration
